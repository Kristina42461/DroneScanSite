О продукте

ДронСкан — программная платформа, которая позволяет рою дронов взаимодействовать между собой и окружением, автономно планировать маршруты и выполнять задачи в условиях ограниченной связи и высокой неопределённости.

Координация роя
Обмен задачами, предотвращение конфликтов, распределение ролей.
Компьютерное зрение
Обнаружение объектов, слежение, семантические карты местности.
  • Python 3.10+
  • djitellopy (DJI Tello)
  • OpenCV
  • PyAV
  • NumPy
  • PyTorch
  • FastAPI / WebSocket

Что мы предлагаем

  • Прототип ПО для симуляции роя

Области применения

Экологический мониторинг

Учёт биоразнообразия, обнаружение загрязнений, мониторинг пожаров и наводнений.

Геология и картография

Съёмка рельефа, 3D-реконструкция, построение высокоточных карт местности.

Поисково-спасательные

Быстрый облёт территории, обнаружение людей, доставка медикаментов.

Экстремальные локации

Работа в зонах риска: горы, пещеры, промобъекты, вулканы, ледники.

Технологии

Сейчас (MVP)
  • Python 3.10+, алгоритмы CBBA‑lite / Hybrid coverage / Energy‑aware RTB
  • Управление учебными дронами: DJI Tello через djitellopy
  • Видео/изображения: OpenCV, PyAV
  • Численные вычисления: NumPy
  • Модели/исследования: PyTorch (библиотека установлена)
Python 3.10+ djitellopy OpenCV PyAV NumPy PyTorch
Дорожная карта (в планах)
  • ROS2 и MAVLink/MAVSDK, поддержка PX4/ArduPilot
  • API и телеметрия: FastAPI + WebSocket; веб‑консоль оператора (React)
  • CV‑модули: детекция/трекер на edge; ONNX Runtime
  • SLAM/VO и семантические карты местности
  • GStreamer для мультимедийного конвейера
ROS2 MAVLink/MAVSDK PX4/ArduPilot FastAPI WebSocket React ONNX Runtime GStreamer
